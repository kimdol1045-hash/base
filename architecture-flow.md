## 1. 전체 흐름 요약

```
사용자 요구사항
     │
     ▼
┌─────────┐    ┌───────────┐    ┌─────────┐    ┌───────────┐
│  분류    │ →  │  Skill    │ →  │  실행   │ →  │  검증     │ → 최종 출력
│ (Haiku)  │    │  조립     │    │(Claude) │    │ (Haiku)   │
└─────────┘    └───────────┘    └─────────┘    └───────────┘
  ~200ms         <1ms           ~2-5s           ~500ms
  ~0.001원       무료           작업별 상이       ~0.01원
```

```
┃                                                                  ┃

┃  └─ user: "PUT /users/:id에 인증 미들웨어가 누락되었습니다.       ┃

┃                                                                  ┃
┃  → 수정된 코드 생성 → 다시 POST 검증                              ┃
┃  → PASS면 반환, FAIL이면 한 번 더 재시도                          ┃
┃  → 3회 실패 시: 출력 + ⚠️ 미통과 항목 안내와 함께 반환             ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━┳━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

                            ▼

╚══════════════════════════════════════════════════════════════════╝
```
