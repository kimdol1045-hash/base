# marketing.copy.headline-formula

> 헤드라인은 광고/콘텐츠의 가장 중요한 요소로, David Ogilvy는 본문 대비 헤드라인을 읽는 사람이 5배 많다고 했다. 검증된 헤드라인 공식을 활용하면 클릭률과 관심도를 크게 향상시킬 수 있다.

# 헤드라인 공식 가이드

## 핵심 원칙
- 7단어 이내가 이상적
- 구체적 숫자 또는 결과 포함
- 독자의 감정(호기심, 두려움, 욕구)을 자극
- 약속한 것을 본문에서 반드시 이행

## 검증된 헤드라인 공식
| 공식 | 예시 |
|------|------|
| 숫자 + 결과 | "전환율을 37% 높이는 5가지 방법" |
| 어떻게 | "어떻게 3개월 만에 10만 사용자를 확보했는가" |
| 왜 ~ 하는가 | "왜 90%의 스타트업이 마케팅에 실패하는가" |
| ~없이 | "코딩 없이 웹사이트를 만드는 법" |
| 비밀 | "탑 셀러들이 사용하는 가격 전략의 비밀" |
| 실수 | "이메일 마케팅에서 흔히 저지르는 7가지 실수" |

## 헤드라인 테스트 기준 (4U)
1. **Useful**: 독자에게 유용한가?
2. **Urgent**: 지금 읽어야 할 이유가 있는가?
3. **Ultra-specific**: 구체적인가?
4. **Unique**: 독특한가?

## A/B 테스트 포인트
- 숫자 포함 vs 미포함
- 긍정 프레임 vs 부정 프레임
- 질문형 vs 진술형
- 짧은 헤드라인 vs 긴 헤드라인

## DO
- 헤드라인 변형 3개 이상 작성 후 선택
- 검색 의도(SEO)와 클릭 욕구(CTR) 모두 고려
- 모바일에서 잘리지 않는 길이 확인

## DON'T
- 클릭베이트 (본문과 불일치) 사용하지 않기
- 모호한 헤드라인 ("알아야 할 것") 사용하지 않기
- 전문 용어로만 구성하지 않기
