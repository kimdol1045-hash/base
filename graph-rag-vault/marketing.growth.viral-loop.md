# marketing.growth.viral-loop

> Viral Loop — K-factor 기반 자기 증식 성장 모델

# 바이럴 루프 설계

## 핵심 원칙
- K-factor = 초대 수 × 전환율 (K > 1이면 바이럴)
- 제품 사용 자체가 공유를 유발하는 구조 설계
- 공유 마찰을 최소화하고 인센티브 정렬

## DO
- 자연스러운 공유 트리거 포인트 식별
- 양면 인센티브 (초대자 + 피초대자 모두 혜택)
- K-factor 측정 및 단계별 최적화

## DON'T
- 스팸성 초대 메커니즘 설계하지 않기
- 제품 가치 없이 인센티브만 의존하지 않기
- 바이럴 지표만 추적하고 리텐션 무시하지 않기
