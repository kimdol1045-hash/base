# marketing.copy.storytelling

> 마케팅 스토리텔링은 Donald Miller의 StoryBrand 프레임워크에 기반하여, 고객을 영웅으로 브랜드를 가이드로 포지셔닝하는 서사 전략이다. 고객이 원하는 것 → 문제 → 가이드 → 계획 → CTA → 성공/실패 구조를 따른다.

# 마케팅 스토리텔링 가이드

## 핵심 원칙
- 고객이 영웅, 브랜드는 가이드
- 고객의 문제에서 시작
- 변화(Before → After)를 보여주기
- 감정적 연결 + 논리적 근거 결합

## StoryBrand 7단계
1. **캐릭터**: 고객이 원하는 것 정의
2. **문제**: 외적 + 내적 + 철학적 문제
3. **가이드**: 브랜드 = 공감 + 권위
4. **계획**: 간단한 3단계 로드맵
5. **행동 촉구**: 명확한 CTA
6. **실패 회피**: 행동하지 않으면 어떻게 되는가
7. **성공**: 행동 후 변화된 모습

## 적용 예시
| 요소 | 예시 (SaaS) |
|------|-----------|
| 캐릭터 | PM이 팀 생산성을 높이고 싶다 |
| 문제 | 산재된 도구, 비효율적 커뮤니케이션 |
| 가이드 | "1,000개 팀이 신뢰하는" + 공감 |
| 계획 | 가입 → 팀 초대 → 첫 프로젝트 |
| CTA | "14일 무료로 시작하기" |
| 실패 | 프로젝트 지연, 팀 소통 단절 |
| 성공 | 생산성 34% 향상, 미팅 50% 감소 |

## DO
- 케이스 스터디를 StoryBrand 구조로 작성
- Before/After를 구체적 수치로 보여주기
- 고객의 언어를 그대로 사용

## DON'T
- 브랜드를 영웅으로 포지셔닝하지 않기
- 문제 제시 없이 솔루션만 나열하지 않기
- 과장된 성공 스토리 사용하지 않기
