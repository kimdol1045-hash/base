# analytics.predictive-analytics

> 예측 분석은 과거 데이터와 통계 모델을 활용하여 미래 결과를 예측하는 분석 기법이다. 회귀 분석, 분류 모델, 시계열 예측이 핵심이며, 비즈니스 의사결정의 선행 정보를 제공한다.

# 예측 분석 가이드

## 핵심 원칙
- 예측의 목적은 정확한 숫자가 아닌 의사결정 지원
- 모델 복잡도와 해석 가능성의 균형
- 예측 구간(Confidence Interval) 반드시 제시
- 모델 성능을 주기적으로 모니터링 (드리프트 감지)

## 주요 예측 유형
| 유형 | 방법 | 활용 |
|------|------|------|
| 수요 예측 | 시계열 (ARIMA, Prophet) | 재고 관리, 리소스 계획 |
| 이탈 예측 | 분류 (XGBoost, LR) | 고객 유지 |
| 매출 예측 | 회귀 + 시계열 | 예산 계획, 목표 설정 |
| 리드 스코어링 | 분류 모델 | 영업 우선순위 |

## 분석 프로세스
1. 비즈니스 문제 정의 → 예측 대상 결정
2. 데이터 수집 및 전처리
3. 피처 엔지니어링
4. 모델 선택 및 학습
5. 검증 (교차 검증, 시간 분할)
6. 배포 및 모니터링

## DO
- 예측값과 함께 신뢰 구간 제시
- 베이스라인 모델과 비교 (단순 평균, 작년 동기)
- 피처 중요도 분석으로 해석 가능성 확보

## DON'T
- 과적합(Overfitting)된 모델을 배포하지 않기
- 학습 데이터와 다른 분포의 데이터에 적용하지 않기
- 모델 예측을 맹신하지 않기 (비즈니스 판단 병행)
