# marketing.growth.inbound

> 인바운드 마케팅은 HubSpot이 체계화한 방법론으로, 가치 있는 콘텐츠를 통해 잠재 고객을 자연스럽게 유인하는 전략이다. Attract → Convert → Close → Delight의 4단계 플라이휠 모델이 핵심이다.

# 인바운드 마케팅 가이드

## 핵심 원칙
- 가치 제공 우선, 판매는 그 다음
- 콘텐츠로 신뢰를 쌓고 자연스러운 전환 유도
- 바이어 페르소나 기반 맞춤 콘텐츠
- 퍼널 단계별 다른 콘텐츠와 CTA

## 인바운드 플라이휠
1. **유인(Attract)**: 블로그, SEO, SNS로 방문자 유입
2. **전환(Convert)**: 랜딩 페이지, 리드 마그넷으로 리드 확보
3. **클로징(Close)**: 이메일 너처링, CRM으로 고객 전환
4. **기쁨(Delight)**: 성공 사례, 커뮤니티로 옹호자 육성

## 퍼널별 콘텐츠 전략
| 단계 | 의도 | 콘텐츠 유형 |
|------|------|-----------|
| 인지(TOFU) | 문제 인식 | 블로그, 인포그래픽 |
| 관심(MOFU) | 솔루션 탐색 | 백서, 웨비나, 케이스 스터디 |
| 결정(BOFU) | 구매 결정 | 데모, 무료 체험, 비교표 |

## 리드 마그넷 유형
- 체크리스트/템플릿, 백서/eBook, 무료 도구
- 웨비나/마스터클래스, ROI 계산기

## DO
- 바이어 페르소나별 콘텐츠 맵 작성
- 리드 스코어링으로 영업 적격 리드 자동 전달
- 콘텐츠 성과를 리드/매출까지 추적

## DON'T
- 게이트 콘텐츠만 남발하지 않기 (비게이트 콘텐츠도 필요)
- 리드 확보 후 방치하지 않기 (너처링 필수)
- 모든 방문자에게 동일한 CTA 제공하지 않기
