# marketing.persuasion.commitment

> **Region**: 📡 [[SENSORS]]  
> **Domain**: `marketing`  
> **Type**: `pattern`  
> **Theory**: #66 일관성 (Cialdini, 1984 Ch.3)  
> **Tokens**: 490

## Content

일관성의 원칙 — 한번 동의하면 그 방향으로 계속 행동하려는 심리:

### 핵심 원리
사람은 자신의 이전 행동/발언과 일관되게 행동하려 한다.
작은 동의(micro-commitment)가 큰 동의(구매, 구독)로 이어진다.
Foot-in-the-Door 기법: 작은 요청 수락 → 큰 요청 수락률 상승.

### 마이크로 커밋먼트 단계 설계

**SaaS 온보딩 퍼널**:
```
Step 1: 이메일 입력 (최소 커밋먼트)
     ↓  "무료로 시작하기"
Step 2: 이름 + 회사명 입력
     ↓  "거의 다 됐습니다!"
Step 3: 역할 + 팀 규모 선택
     ↓  "맞춤 설정 중..."
Step 4: 첫 프로젝트 생성 (가치 체험)
     ↓  "프로젝트가 준비되었습니다!"
Step 5: 팀원 초대 (사회적 커밋먼트)
     ↓  "팀원이 합류했습니다"
Step 6: 유료 전환 제안
     ↓  "프로로 업그레이드하면 팀 전체가..."
```

**각 단계의 커밋먼트 심리**:
| 단계 | 투자 유형 | 일관성 압력 |
|------|----------|-----------|
| 이메일 입력 | 최소 정보 | "이미 시작했으니 계속해야지" |
| 프로필 작성 | 시간 + 개인정보 | "여기까지 했는데 그만둘 수 없지" |
| 데이터 입력 | 노력 + 데이터 | "이 데이터를 버릴 수 없지" |
| 팀원 초대 | 사회적 관계 | "팀원도 쓰고 있는데" |
| 커스터마이징 | 시간 + 개인화 | "내 것으로 만들었는데" |

### 프로그레스 바 활용
진행 상황 표시로 완성 욕구를 자극한다 (Zeigarnik Effect와 결합):
```
┌─────────────────────────────────────┐
│  프로필 완성도: 60%                   │
│  ████████████░░░░░░░░               │
│  다음 단계: 프로필 사진 추가            │
│  [완성하기]                           │
└─────────────────────────────────────┘
```

### 공개 선언 활용
- "목표를 설정하세요" → 입력한 목표가 대시보드에 표시
- "올해 목표: 매출 2배 달성" → 매주 진행률 리마인더
- SNS 공유: "OOO 챌린지에 참여합니다" → 공개적 커밋먼트

### 한국어 카피 예시 (각 단계별)
- Step 1: "이메일만 입력하면 바로 시작합니다" (진입 장벽 최소화)
- Step 2: "30초만 투자하세요. 맞춤 추천을 드립니다" (소요 시간 명시)
- Step 3: "거의 다 왔습니다! 마지막 한 단계" (완성 욕구)
- Step 4: "첫 프로젝트를 만들어보세요 — 1분이면 됩니다" (가치 체험)
- Step 5: "팀원을 초대하면 협업이 시작됩니다" (사회적 커밋먼트)
- 해지 방어: "지금까지 47개 프로젝트를 생성하셨습니다" (투자 상기)

### 설계 원칙
- 각 단계는 이전 단계보다 약간만 더 큰 요청이어야 한다
- 각 단계마다 즉각적인 가치를 제공해야 한다 (일방적 요청 금지)
- 프로그레스 바로 진행 상황을 시각화한다
- 이전 입력 데이터를 활용한 맞춤 경험을 제공한다
