## Content

Content SEO — 검색 의도에 맞는 콘텐츠를 구조적으로 작성하여 검색 순위와 사용자 만족 동시 달성:

### 키워드 리서치 통합 프로세스
```
1. 시드 키워드 도출 (제품/서비스 핵심 기능)
2. 검색 의도 분류:
   - Informational: "프로젝트 관리 방법" → 블로그/가이드
   - Navigational: "AppName 로그인" → 로그인 페이지
   - Commercial: "프로젝트 관리 도구 비교" → 비교 페이지
   - Transactional: "프로젝트 관리 도구 가격" → 가격 페이지
3. 키워드 난이도 vs 검색량 매트릭스:
   - 초기: 롱테일 (월 100~500, 난이도 < 30)
   - 성장기: 미드테일 (월 500~5,000, 난이도 < 50)
   - 성숙기: 헤드 키워드 (월 5,000+, 난이도 < 70)
```

### 헤딩 계층 구조 (H1 → H2 → H3)
```html
<!-- DO: 논리적 헤딩 계층, 키워드 자연 배치 -->
<h1>팀 프로젝트 관리를 위한 완벽 가이드 [2024]</h1>  <!-- 1개만 -->
  <h2>프로젝트 관리란 무엇인가?</h2>
    <h3>프로젝트 관리의 5가지 핵심 요소</h3>
    <h3>애자일 vs 워터폴 방법론 비교</h3>
  <h2>프로젝트 관리 도구 선택 기준</h2>
    <h3>팀 규모별 추천 도구</h3>
    <h3>가격 대비 기능 비교</h3>
  <h2>프로젝트 관리 도구 도입 5단계</h2>
```

DON'T:
```html
<!-- ❌ H1 중복 — 검색 엔진 혼란 -->
<h1>프로젝트 관리</h1>
<h1>최고의 프로젝트 관리 도구</h1>

<!-- ❌ 헤딩 레벨 건너뛰기 -->
<h1>가이드</h1>
<h3>첫 번째 섹션</h3>  <!-- h2 건너뜀 -->

<!-- ❌ 스타일링 목적의 헤딩 사용 -->
<h3>※ 주의사항</h3>  <!-- CSS로 해결, 시맨틱 오용 금지 -->
```

### 키워드 배치 전략 (자연스러움 우선)
```
┌─────────────────────────────────────────────┐
│ 1. Title 태그 — 키워드 앞쪽 배치            │  ← 가장 중요
│ 2. H1 태그 — 메인 키워드 포함               │
│ 3. 첫 100단어 — 핵심 키워드 1회 자연 삽입    │  ← 크롤러 우선 확인
│ 4. H2/H3 — 관련 키워드 / LSI 키워드         │
│ 5. 이미지 alt text — 설명적 + 키워드         │
│ 6. URL slug — 짧고 키워드 포함               │
│ 7. 마지막 문단 — 키워드 자연 재등장           │
└─────────────────────────────────────────────┘
키워드 밀도: 전체 텍스트의 1~2% (강제 삽입 금지)
```

### 내부 링크 전략
```
┌──────────────┐
│   홈 (/)     │ ← 모든 주요 페이지로 링크
├──────┬───────┤
│ /features  /pricing │ ← 허브 페이지 (pillar content)
├──────┼───────┤
│ /blog/1  /blog/2   │ ← 클러스터 콘텐츠 → 허브로 링크
└──────┴───────┘

규칙:
- 모든 페이지는 홈에서 3클릭 이내 도달 가능
- 앵커 텍스트는 설명적 (DO: "프로젝트 관리 비교", DON'T: "여기를 클릭")
- 신규 콘텐츠 발행 시 기존 관련 콘텐츠에 링크 추가
- 고아 페이지 (어디서도 링크되지 않는 페이지) 제거
```

DON'T:
```
❌ 키워드 스터핑:
"프로젝트 관리 도구를 찾으시나요? 프로젝트 관리 도구는
 프로젝트 관리를 위한 프로젝트 관리 도구입니다."
→ Google Spam 정책 위반 → 순위 하락 또는 페널티

❌ "여기를 클릭하세요" 앵커 텍스트:
<a href="/pricing">여기를 클릭하세요</a>
→ 검색 엔진에게 링크 맥락 정보 제로

❌ 콘텐츠 방치 (freshness decay):
2년 전 작성 후 업데이트 없음 → 순위 점진적 하락
→ 핵심 콘텐츠는 분기 1회 업데이트 (날짜 갱신 + 최신 정보 추가)
```
