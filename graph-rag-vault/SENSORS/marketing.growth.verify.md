## Content

Growth 마케팅 검증 체크리스트 — 성장 전략의 핵심 요소를 체계적으로 점검:

### 랜딩 페이지 체크리스트
- [ ] CTA가 above-the-fold (스크롤 없이 보이는 영역)에 위치하는가?
- [ ] CTA가 1개로 명확하고 "행동 동사 + 결과"를 포함하는가?
- [ ] 헤드라인이 기능이 아닌 이점(Benefit) 중심인가?
- [ ] 소셜 프루프가 포함되었는가? (숫자, 로고, 후기 중 최소 2종)
- [ ] 소셜 프루프가 CTA 근처에 배치되었는가?
- [ ] 리스크 제거 문구가 있는가? (무료 체험, 카드 불필요, 언제든 취소)
- [ ] 페이지 로딩 시간이 3초 이내인가? (1초 지연 = 전환율 7% 하락)
- [ ] 모바일 환경에서 레이아웃이 깨지지 않고 CTA 터치 영역이 44px 이상인가?
- [ ] A/B 테스트 계획이 수립되었는가? (헤드라인, CTA, 소셜 프루프)

### 이메일 시퀀스 체크리스트
- [ ] 온보딩 이메일 시퀀스가 설계되었는가? (Welcome → Tutorial → Value → Upgrade)
- [ ] 각 이메일에 단일 CTA가 있는가? (CTA 2개 이하)
- [ ] 제목이 30~50자이고 개인화([이름])를 포함하는가?
- [ ] 모든 마케팅 이메일에 수신 거부(unsubscribe) 링크가 있는가?
- [ ] 비활성 사용자 재활성화 시퀀스가 있는가?
- [ ] 이메일 발송 빈도가 하루 1통 이하인가?

### AARRR 퍼널 체크리스트
- [ ] Acquisition: 주요 유입 채널 3개 이상 확보하였는가?
- [ ] Activation: Aha Moment까지의 단계가 정의되었는가?
- [ ] Activation: 가입 → 핵심 행동 전환율이 40% 이상인가?
- [ ] Retention: D1 리텐션 > 40%, W1 > 35% 기준을 충족하는가?
- [ ] Revenue: LTV/CAC 비율이 3배 이상인가?
- [ ] Referral: 레퍼럴/초대 기능이 구현되었는가?
- [ ] 각 AARRR 단계별 측정 이벤트가 Analytics에 설정되었는가?

### 소셜 미디어 체크리스트
- [ ] Open Graph 태그가 설정되어 링크 공유 시 미리보기가 적절한가?
- [ ] OG 이미지가 1200x630px 이상이고 텍스트가 포함되었는가?
- [ ] 플랫폼별 네이티브 포맷으로 콘텐츠가 제작되었는가?
- [ ] 일관된 게시 스케줄이 수립되었는가?

### 그로스 실험 체크리스트
- [ ] 가설이 "[변경]하면 [지표]가 [방향]할 것이다" 형식인가?
- [ ] A/B 테스트 표본 크기와 기간이 산출되었는가? (통계적 유의성 95%)
- [ ] ICE 스코어 (Impact x Confidence x Ease)로 우선순위가 정해졌는가?
- [ ] 주 2~3개 실험 사이클이 운영 중인가?
- [ ] North Star Metric이 정의되고 팀 전체가 공유하고 있는가?
