# marketing.persuasion.cognitive-dissonance

> **Region**: 📡 [[SENSORS]]  
> **Domain**: `marketing`  
> **Type**: `rule`  
> **Theory**: #68 인지 부조화 (Festinger, 1957)  
> **Tokens**: 490

## Content

인지 부조화 — 행동과 신념이 불일치할 때 심리적 불편함을 해소하려 한다:

### 핵심 원리
사람은 자신의 행동과 태도가 모순될 때 불편함을 느낀다.
이를 해소하기 위해 (1) 태도를 바꾸거나 (2) 행동을 정당화한다.
마케팅에서는 "이미 투자한 시간/노력"이 이탈을 막는 장치가 된다.

### 적용 전략

**1. 온보딩에서 투자 유도 → 이탈 비용 증가**
사용자가 초기에 시간과 데이터를 투입하면, "여기까지 했는데 그만둘 수 없다"는 심리 발생.

| 투자 유형 | 구현 방법 | 이탈 비용 |
|----------|----------|----------|
| 데이터 입력 | 프로필, 선호도 설문, 목표 설정 | 맞춤 설정 소실 |
| 콘텐츠 생성 | 문서 작성, 프로젝트 설정 | 생성물 소실 |
| 연동 설정 | API 연동, 데이터 임포트 | 재설정 비용 |
| 학습 투자 | 사용법 학습, 단축키 습득 | 재학습 비용 |
| 사회적 투자 | 팀원 초대, 공유 설정 | 팀 이전 비용 |

**온보딩 퍼널 설계 예시**:
```
가입 → "맞춤 추천을 위해 3가지만 알려주세요"
     → 업종 선택, 팀 규모, 주요 목표
     → "김민수님에게 최적화된 대시보드를 준비했습니다"
     → 대시보드 커스터마이징 (위젯 배치, 색상)
     → 데이터 연동 (구글 시트, Slack)
     → 이제 사용자는 "내 것"이 된 도구를 쉽게 떠나지 못한다
```

**2. 구매 후 인지 부조화 해소 (Post-Purchase)**
구매 직후 "이게 맞는 선택이었나?" 불안 → 정당화 지원.
- 확인 이메일: "현명한 선택을 하셨습니다. 이 플랜을 사용하는 기업들:"
- 성공 사례 제공: "같은 플랜을 선택한 ABC사는 3개월 만에 ROI 340% 달성"
- 빠른 성과 유도: 가입 후 24시간 내 "첫 번째 성과"를 경험하게 설계

**3. 해지 방어에서 투자 상기**
```
┌─────────────────────────────────────┐
│  해지 전 확인해주세요                  │
│                                      │
│  지난 6개월간 활동 요약:              │
│  - 생성한 프로젝트: 47개             │
│  - 협업한 팀원: 12명                 │
│  - 절약한 시간: 약 156시간           │
│  - 연동된 서비스: 5개                │
│                                      │
│  해지하시면 이 데이터에 접근할 수       │
│  없게 됩니다.                         │
│                                      │
│  [플랜 유지하기]  [그래도 해지하기]     │
└─────────────────────────────────────┘
```

### 한국어 카피 예시
- "이미 80% 완성하셨습니다. 마지막 단계만 남았어요"
- "지금까지 입력하신 데이터를 기반으로 맞춤 분석이 준비되었습니다"
- "김민수님만의 대시보드가 완성되었습니다"
- "팀원 5명이 이미 협업 중입니다. 정말 해지하시겠습니까?"

### 윤리적 경계
- 해지를 의도적으로 어렵게 만들지 않는다 (다크 패턴)
- 투자 사실을 객관적으로 보여주되, 죄책감을 유발하지 않는다
- 사용자가 원하면 언제든 데이터 내보내기를 제공한다
