# marketing.growth.plg

> PLG(Product-Led Growth)는 제품 자체가 사용자 획득, 활성화, 전환, 확장의 주요 동력이 되는 성장 전략이다. Slack, Figma, Notion이 대표적 PLG 기업이며, 무료 체험과 셀프 서비스가 핵심이다.

# PLG(Product-Led Growth) 가이드

## 핵심 원칙
- 제품이 최고의 마케팅 도구
- 가치 먼저 제공, 결제는 나중에
- 셀프 서비스 온보딩으로 영업 비용 최소화
- 바이럴 루프를 제품에 내장

## PLG 퍼널
1. **획득**: 무료 체험/프리미엄으로 진입 장벽 제거
2. **활성화**: Time to Value 최소화 (첫 가치 경험)
3. **전환**: 사용량 기반 유료 전환 트리거
4. **확장**: 팀/조직 단위 확산 (Bottom-up)
5. **추천**: 제품 내 공유/초대 기능

## PLG 핵심 지표
| 지표 | 설명 |
|------|------|
| Time to Value | 가입~첫 가치 경험 시간 |
| PQL (Product Qualified Lead) | 제품 사용 기반 적격 리드 |
| 자연 이탈률 | 무료 → 유료 전환 없이 이탈 |
| 바이럴 계수 | 사용자당 초대 수 |
| 확장 매출(NRR) | 기존 고객 매출 성장률 |

## DO
- 가입 후 5분 내 핵심 가치 경험 가능하게 설계
- PQL 기준을 정의하고 세일즈에 자동 전달
- 프리미엄 경계를 사용자 가치 기준으로 설정

## DON'T
- 무료 버전의 가치를 과도하게 제한하지 않기
- 온보딩에 영업 전화를 강제하지 않기
- 가격 페이지를 숨기지 않기 (투명성)
