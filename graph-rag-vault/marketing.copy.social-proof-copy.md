# marketing.copy.social-proof-copy

> #62 Social Proof (Cialdini, 1984)

소셜 프루프 카피 (다른 사람이 쓴다는 증거로 신뢰를 구축한다):

### 유형별 활용
1. **숫자**: "10,000+ 개발자가 사용 중" — 가장 직관적
2. **로고월**: 고객사 로고 5-8개 나열 (알려진 브랜드 우선)
3. **후기/추천사**: 실명+직책+사진 포함. 구체적 결과 언급.
   - DO: "배포 시간이 3시간에서 15분으로 줄었습니다. — 김OO, OO사 CTO"
   - DON'T: "좋은 제품입니다. — 익명"
4. **사례 연구**: Before→After 수치 중심
   "도입 전 주 3회 장애 → 도입 후 월 0회"
5. **미디어 노출**: "~에 소개되었습니다" + 미디어 로고
6. **실시간 증거**: "지금 42명이 보고 있습니다" (과하면 조작처럼 보임)

### 배치 위치
- 히어로 섹션 아래 (첫 신뢰 구축)
- 가격 페이지 (결정 직전)
- 가입 폼 옆 (전환 포인트)

### 한국어 카피 예시
- "누적 가입자 50,000명 돌파"
- "고객 평점 4.9/5.0 (1,200개 리뷰)"
- "국내 스타트업 300곳이 선택한 이유"
