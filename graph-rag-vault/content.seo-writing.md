# content.seo-writing

> SEO 라이팅은 검색 엔진 최적화와 사용자 가독성을 동시에 만족시키는 글쓰기 기법이다. Google의 Helpful Content Update 이후 사용자 의도(Search Intent) 매칭이 키워드 밀도보다 중요해졌다.

# SEO 라이팅 가이드

## 핵심 원칙
- 검색 의도(Intent) 우선: 정보형/거래형/탐색형 구분
- 키워드는 자연스럽게 배치 (밀도 1-2%)
- 사용자 가독성이 크롤러 최적화보다 우선
- E-E-A-T (경험, 전문성, 권위, 신뢰) 신호 강화

## 작성 프로세스
1. 키워드 리서치: 검색량 + 난이도 + 의도 분석
2. 경쟁 콘텐츠 분석: 상위 10개 결과 구조 파악
3. 아웃라인 작성: H2/H3 구조에 키워드 배치
4. 본문 작성: 역피라미드 구조, 짧은 문단
5. 메타 데이터: 타이틀 60자, 디스크립션 155자

## DO
- 타이틀 태그 앞부분에 주요 키워드 배치
- H2/H3에 롱테일 키워드 자연스럽게 포함
- 내부 링크 3-5개, 외부 권위 링크 1-2개 삽입
- FAQ 섹션 추가로 구조화된 데이터 확보

## DON'T
- 키워드 스터핑 (동일 키워드 과다 반복)
- 중복 콘텐츠 생성 (카니발라이제이션)
- 클릭베이트 타이틀로 이탈률 높이지 않기
- AI 생성 콘텐츠를 검수 없이 발행하지 않기
