# meta.role-prompting

> 역할 부여 프롬프팅 — 페르소나 기반 응답 품질 향상

# Role Prompting 가이드

## 핵심 원칙
- LLM에 특정 전문가 역할을 부여하여 응답 품질 향상
- 역할의 전문성, 경험, 사고방식을 명시
- 도메인 특화 어휘와 관점을 유도

## DO
- 역할의 구체적 경력과 전문 분야 명시
- 역할에 맞는 제약 조건 설정
- 복수 역할 토론 형식으로 다각적 분석

## DON'T
- 모호한 역할 (예: "전문가") 부여하지 않기
- 역할과 무관한 작업 요청하지 않기
- 역할 간 충돌하는 지시 동시 부여하지 않기
