# marketing.growth.ab-testing-marketing

> 마케팅 A/B 테스트 — 데이터 기반 마케팅 최적화

# 마케팅 A/B 테스트 가이드

## 핵심 원칙
- 단일 변수 격리 테스트로 인과관계 확인
- 통계적 유의성 확보 후 의사결정
- 테스트 문화 정착으로 지속적 최적화

## DO
- 가설 → 실험 → 분석 → 적용 사이클 운영
- MDE(최소 감지 효과) 기반 샘플 사이즈 산정
- 랜딩 페이지, CTA, 카피, 이미지 순차 테스트

## DON'T
- 동시에 여러 변수 변경하지 않기
- 유의성 없이 조기 종료하지 않기
- 시즌/이벤트 기간 중 테스트 결과 일반화하지 않기
