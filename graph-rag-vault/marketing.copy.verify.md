# marketing.copy.verify

마케팅 카피 자기 검증 체크리스트 — 모든 항목을 통과해야 출력한다.

### A. 3초 테스트
- [ ] 헤드라인만 읽고 3초 내에 "이게 뭔지, 나한테 왜 필요한지" 파악 가능한가?
- [ ] 헤드라인이 7단어 이내인가?
- [ ] 헤드라인에 구체적 숫자 또는 고통 포인트가 포함되어 있는가?
- [ ] 스크롤 없이 핵심 가치 제안이 보이는가? (Above the Fold)

### B. AIDA 구조
- [ ] Attention: 호기심 또는 고통 포인트로 주의를 끌고 있는가?
- [ ] Interest: 기능(Feature)이 아닌 이점(Benefit)을 제시하고 있는가?
- [ ] Desire: 사회적 증거(후기/수치/로고)가 최소 1가지 포함되어 있는가?
- [ ] Action: 구체적 CTA가 명확히 존재하는가?
- [ ] AIDA 순서를 건너뛴 단계가 없는가?

### C. 설득 경로 (ELM)
- [ ] 타겟의 관여도를 명시적으로 판단했는가? (고관여/저관여)
- [ ] 고관여: 데이터, 비교, ROI 등 논리적 근거를 제시했는가?
- [ ] 저관여: 감정, 사회적 증거, 시각 자극을 활용했는가?
- [ ] 경로 선택이 타겟과 제품에 적합한가?

### D. CTA 품질
- [ ] CTA 문구가 "행동 동사 + 구체적 결과" 형식인가?
- [ ] 주요 CTA가 페이지에 1개로 명확한가?
- [ ] CTA 아래에 리스크 제거 문구가 있는가? (무료, 취소 가능 등)
- [ ] "클릭하세요", "자세히 보기" 같은 모호한 CTA가 아닌가?
- [ ] 긴급성/희소성 표현이 진실한가? (거짓 타이머, 과장 금지)

### E. 카피 품질
- [ ] 한 문장이 20자 이내인가? (긴 문장 분리)
- [ ] 한 문단이 3줄 이내인가?
- [ ] 전문 용어 없이 쉽게 읽히는가? (초6 수준)
- [ ] 수동태가 아닌 능동태를 사용했는가?
- [ ] 추상적 표현 대신 구체적 수치/사례가 있는가?

### F. 윤리 검증
- [ ] 과장되거나 검증 불가능한 주장이 없는가? ("업계 최고" 등)
- [ ] 거짓 긴급성/희소성이 없는가?
- [ ] 경쟁사를 부정적으로 언급하지 않았는가?
- [ ] 소비자 보호법/광고법 위반 소지가 없는가?
- [ ] 후기/추천이 실제 또는 합리적 가상인가? (날조 금지)

### G. SEO (해당 시)
- [ ] 타겟 키워드가 헤드라인/서브헤드에 자연스럽게 포함되어 있는가?
- [ ] 메타 디스크립션이 155자 이내이며 CTA를 포함하는가?
- [ ] H1 → H2 → H3 계층이 논리적인가?
