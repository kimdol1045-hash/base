# marketing.copy.headline

> #57 Ogilvy Advertising Principles (Ogilvy, 1963)

헤드라인 공식 (독자의 80%는 헤드라인만 읽는다 — Ogilvy):

### 검증된 공식 8가지

1. **숫자 공식**: "개발 생산성을 2배로 높이는 5가지 방법"
2. **How-to**: "초보자도 10분 만에 배포하는 방법"
3. **질문형**: "아직도 수동으로 테스트하고 계신가요?"
4. **네거티브**: "절대 하지 말아야 할 API 설계 실수 7가지"
5. **비교형**: "REST vs GraphQL: 당신의 프로젝트에 맞는 선택"
6. **사회적 증거**: "500개 스타트업이 선택한 배포 자동화 도구"
7. **시간 한정**: "이번 주까지만 무료로 사용 가능합니다"
8. **결과 제시**: "도입 2주 만에 배포 시간 80% 단축"

### 작성 규칙
- 30자 이내 (한국어 기준)
- 구체적 숫자 포함 (3배, 5가지, 10분)
- 독자의 이익을 담는다 (기능 ❌ → 결과 ✅)
- 한 번에 10개 작성 후 최고를 선택

### DON'T
- "최고의 도구를 소개합니다" (모호, 근거 없음)
- "혁신적인 솔루션" (의미 없는 형용사)
- "우리 제품은 ~합니다" (독자 관점 아닌 제품 관점)
