# marketing.persuasion.prospect-theory

> #61 프로스펙트 이론 (Kahneman & Tversky, 1979)

프로스펙트 이론 — 손실의 고통은 이득의 기쁨보다 약 2배 강하다:

### 핵심 원리
1. **손실 회피 (Loss Aversion)**: 10만원 잃는 고통 > 10만원 얻는 기쁨 (약 2~2.5배)
2. **확실성 효과**: 확실한 소이득 > 불확실한 대이득을 선호
3. **준거점 의존**: 절대 가치가 아닌 기준점 대비 변화로 판단

### 손실 프레이밍 vs 이득 프레이밍

| 이득 프레이밍 (약함) | 손실 프레이밍 (강함) |
|--------------------|-------------------|
| "매월 ₩50만원을 절약하세요" | "매월 ₩50만원을 잃고 계십니다" |
| "전환율을 3배 높이세요" | "전환율 기회를 놓치고 있습니다" |
| "무료 체험을 시작하세요" | "14일 무료 체험이 곧 만료됩니다" |
| "프리미엄 기능을 사용하세요" | "프리미엄 기능 접근 권한이 사라집니다" |

### 한국어 카피 예시 — 손실 회피 활용

**무료 체험 전환 유도**:
- "무료 체험 종료까지 3일. 지금까지 만든 프로젝트 5개가 비활성화됩니다"
- "14일간 쌓아온 데이터와 설정, 유지하시겠습니까?"
- "무료 체험 중 절약한 시간: 약 23시간. 다시 수동으로 돌아가시겠습니까?"

**구독 해지 방어**:
- "해지하시면 팀원 12명의 협업 공간이 삭제됩니다"
- "지난 6개월간 작성한 문서 147개에 더 이상 접근할 수 없습니다"
- "현재 가격은 얼리어답터 할인가입니다. 재가입 시 정가가 적용됩니다"

**한정 할인/프로모션**:
- "이 가격은 오늘 자정에 사라집니다"
- "지금 놓치면 다음 할인은 6개월 후입니다"
- "이미 2,847명이 등록했습니다. 남은 자리 153석"

**기능 업그레이드 유도**:
- "무료 플랜에서는 월 100건까지만 분석 가능. 이번 달 87건 사용하셨습니다"
- "현재 용량 사용률 92%. 파일 업로드가 곧 제한됩니다"
- "경쟁사는 이미 이 기능을 사용하고 있습니다"

**커머스 전환**:
- "장바구니에 담긴 상품의 재고가 5개 남았습니다"
- "오늘 구매하지 않으면 ₩15,000 할인이 소멸됩니다"
- "이 제품을 본 127명 중 34명이 이미 구매했습니다"

**이메일 마케팅**:
- 제목: "김민수님, 놓치고 계신 3가지" (vs "새로운 3가지 기능")
- 제목: "내일이면 사라지는 혜택" (vs "특별 혜택 안내")
- 제목: "회원님만 아직 사용하지 않고 있습니다" (vs "새 기능을 사용해보세요")

### 확실성 효과 활용
- "환불 보장" → 확실한 안전장치 제공으로 구매 결정 촉진
- "결과가 없으면 100% 환불" > "최대 300% ROI 가능"
- "첫 달 무료" > "연간 20% 할인" (확실한 소이득 > 불확실한 대이득)

### 주의: 윤리적 경계
- 실제 데이터 기반 손실만 언급 (허위 긴급성 금지)
- 해지 페이지에서 죄책감 유발 문구 금지 ("정말 떠나실 건가요?" 등)
- 사용자가 실제로 잃는 것이 있을 때만 손실 프레이밍 사용
