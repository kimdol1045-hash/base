# marketing.copy.objection-handling

> 이의 처리(Objection Handling)는 잠재 고객의 구매 저항을 사전에 파악하고 카피에서 해소하는 기법이다. 가격, 신뢰, 필요성, 긴급성, 권한의 5대 이의를 체계적으로 다루어 전환율을 높인다.

# 이의 처리 가이드

## 핵심 원칙
- 이의를 숨기지 말고 먼저 꺼내서 해소
- 5대 이의: 가격, 신뢰, 필요성, 긴급성, 권한
- 이의 해소 순서: 인정 → 공감 → 근거 → 전환
- 사회적 증거가 가장 강력한 이의 해소 도구

## 5대 이의와 해소 전략
| 이의 | 고객 심리 | 해소 전략 |
|------|----------|----------|
| 가격 | "비싸다" | ROI 계산, 비용 대비 효과, 분할 결제 |
| 신뢰 | "믿을 수 있나" | 후기, 사례, 보증, 보안 인증 |
| 필요성 | "지금 필요 없다" | 문제의 비용 가시화, 경쟁사 위험 |
| 긴급성 | "나중에 해도 된다" | 시간 제한, 기회비용, 조기 혜택 |
| 권한 | "내가 결정할 수 없다" | 의사결정자용 요약, ROI 보고서 |

## 카피 적용 패턴
### FAQ 섹션
- "정말 효과가 있나요?" → 구체적 사례 + 수치
- "해지가 어렵지 않나요?" → 1클릭 해지 + 환불 정책

### 가격 페이지
- ROI 계산기 제공
- "하루 3,000원으로" 리프레이밍
- "30일 무조건 환불" 보증

## DO
- 고객 인터뷰에서 실제 이의 수집
- 랜딩 페이지에 이의 해소 섹션 포함
- 사회적 증거(후기, 숫자)를 이의 해소에 활용

## DON'T
- 이의를 무시하고 장점만 나열하지 않기
- 허위 긴급성으로 압박하지 않기
- 환불/해지 정책을 숨기지 않기
