# content.content-metrics

> 콘텐츠 성과 측정은 허영 지표(Vanity Metrics)를 넘어 비즈니스 영향을 정량화하는 프로세스이다. 소비(Consumption), 공유(Sharing), 리드 생성(Lead Gen), 매출(Sales) 4단계로 측정한다.

# 콘텐츠 성과 측정 가이드

## 핵심 원칙
- 비즈니스 목표에 연결된 지표만 추적
- 허영 지표(페이지뷰)와 실행 지표(전환율) 구분
- 콘텐츠 유형별 다른 KPI 적용
- 코호트 분석으로 장기 효과 측정

## 4단계 측정 프레임워크
| 단계 | 지표 | 도구 |
|------|------|------|
| 소비 | 페이지뷰, 체류시간, 스크롤 깊이 | GA4 |
| 참여 | 공유 수, 댓글, 북마크 | SNS API |
| 리드 | 다운로드, 구독, 폼 제출 | CRM |
| 매출 | 어시스트 전환, 직접 전환 | 어트리뷰션 |

## 콘텐츠 유형별 핵심 KPI
- **블로그**: 유기적 트래픽, 체류시간, CTA 클릭률
- **뉴스레터**: 오픈율, 클릭률, 구독 해지율
- **SNS**: 도달률, 참여율, 팔로워 성장률
- **백서**: 다운로드 수, 리드 전환율

## DO
- UTM 파라미터로 채널별 성과 추적
- 월간 콘텐츠 성과 리포트 발행
- 상위 10% / 하위 10% 콘텐츠 분석
- 콘텐츠 ROI = (매출 기여 - 제작 비용) / 제작 비용

## DON'T
- 페이지뷰만으로 성공을 판단하지 않기
- 채널 간 지표를 직접 비교하지 않기 (맥락 다름)
- 단기 성과만 보고 에버그린 콘텐츠 가치를 무시하지 않기
