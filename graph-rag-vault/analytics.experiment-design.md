# analytics.experiment-design

> 실험 설계는 가설을 검증하기 위해 통제된 조건에서 변수를 조작하고 결과를 측정하는 방법론이다. A/B 테스트가 대표적이며, 통계적 유의성(p-value), 검정력(power), 표본 크기 계산이 핵심이다.

# 실험 설계 가이드

## 핵심 원칙
- 가설 → 지표 → 표본 크기 → 실행 → 분석 순서
- 한 번에 하나의 변수만 변경
- 사전에 성공 기준과 종료 조건 정의
- 통계적 유의성 확보 전 조기 종료 금지

## 실험 설계 프로세스
1. **가설 수립**: "X를 Y로 변경하면 Z가 N% 개선될 것"
2. **지표 선정**: 주요 지표 1개 + 가드레일 지표 2-3개
3. **표본 크기 계산**: MDE, 유의수준, 검정력 기반
4. **무작위 배정**: 사용자 단위 해시 기반 분배
5. **실행**: 최소 2주 (요일 효과 포함)
6. **분석**: 유의성 검정 + 세그먼트별 효과 분석

## 표본 크기 산출 요소
| 요소 | 일반 설정 |
|------|----------|
| 유의수준 (alpha) | 0.05 |
| 검정력 (1-beta) | 0.80 |
| 최소 감지 효과 (MDE) | 비즈니스 맥락별 결정 |
| 기본 전환율 | 현재 전환율 |

## DO
- 실험 전 가설과 분석 계획을 문서화
- 가드레일 지표로 부작용 모니터링
- 결과를 실험 저장소에 축적

## DON'T
- p-value가 0.05 미만이 될 때까지 실험 연장하지 않기
- 표본 크기 계산 없이 실험 시작하지 않기
- 여러 변수를 동시에 변경하지 않기
