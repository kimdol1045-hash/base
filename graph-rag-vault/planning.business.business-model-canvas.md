# planning.business.business-model-canvas

> 비즈니스 모델 캔버스(BMC)는 Alexander Osterwalder가 개발한 1페이지 비즈니스 모델 설계 도구이다. 9개 빌딩 블록으로 비즈니스의 가치 창출, 전달, 포착 구조를 시각화한다.

# 비즈니스 모델 캔버스 가이드

## 핵심 원칙
- 1페이지에 비즈니스 모델 전체를 시각화
- 9개 블록 간의 연결과 일관성이 핵심
- 가설 기반: 각 블록의 가정을 명시하고 검증
- 반복 업데이트: 학습에 따라 지속 개선

## 9개 빌딩 블록
| 블록 | 핵심 질문 |
|------|----------|
| 고객 세그먼트 | 누구를 위한 가치인가? |
| 가치 제안 | 어떤 문제를 해결하는가? |
| 채널 | 어떻게 고객에게 전달하는가? |
| 고객 관계 | 어떤 관계를 맺는가? |
| 수익원 | 어떻게 돈을 버는가? |
| 핵심 자원 | 무엇이 필요한가? |
| 핵심 활동 | 무엇을 해야 하는가? |
| 핵심 파트너 | 누구와 협력하는가? |
| 비용 구조 | 비용은 어떻게 발생하는가? |

## 작성 순서
1. 고객 세그먼트 → 가치 제안 (핵심)
2. 채널 → 고객 관계 (전달)
3. 수익원 (수익 모델)
4. 핵심 자원 → 핵심 활동 → 핵심 파트너 (인프라)
5. 비용 구조 (경제성)

## DO
- 가정을 색깔로 표시 (검증됨/미검증)
- 경쟁사 BMC와 비교 분석
- 분기별 BMC 업데이트 워크숍

## DON'T
- 한 번 만들고 방치하지 않기
- 모든 블록을 동시에 검증하려 하지 않기
- BMC를 사업 계획서의 대체물로 사용하지 않기
