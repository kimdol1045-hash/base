# marketing.persuasion.mere-exposure

> #70 Mere Exposure Effect (Zajonc, 1968)

단순 노출 효과 — 반복 노출만으로 호감도가 증가한다:

### 핵심 원리
- 사람은 익숙한 것에 호감을 느낀다 (10-20회 노출이 최적)
- 의식적 인지 없이도 작동 (subliminal exposure도 유효)
- 과다 노출 시 역효과 (wear-out effect) → 변형을 주면서 반복

### 마케팅 적용 패턴
| 채널 | 최적 빈도 | 변형 전략 |
|------|----------|----------|
| 리타겟팅 광고 | 7-12회/월 | 크리에이티브 3-5개 로테이션 |
| 이메일 | 2-4회/월 | 제목/CTA 변형, 동일 브랜드 톤 |
| SNS | 1회/일 | 포맷 변형 (이미지/영상/텍스트) |
| 인앱 | 세션당 1-2회 | 위치/문구 변형, 동일 CTA |

### 브랜드 일관성 + 변형의 균형
```
DO:
- 로고, 색상, 톤은 일관 유지 (브랜드 인식 강화)
- 메시지/크리에이티브만 변형 (wear-out 방지)
- 리타겟팅: 7일 윈도우, frequency cap 12회
- 크로스 채널 노출 (웹 → 이메일 → SNS → 인앱)

DON'T:
- 동일 광고 20회+ 반복 (역효과, 짜증 유발)
- 매번 완전히 다른 디자인 (브랜드 인식 불가)
- frequency cap 없는 리타겟팅 (stalker 느낌)
- 옵트아웃 없는 반복 노출 (윤리적 문제)
```

### 제품 내 적용
- 온보딩: 핵심 기능을 여러 맥락에서 자연스럽게 노출
- 업그레이드 유도: 무료 사용 중 프리미엄 기능 미리보기 반복
- 습관 형성: 같은 시간대에 알림 → 루틴화

### 주의사항
- 노출과 강제는 다름 — 사용자에게 항상 닫기/건너뛰기 제공
- 개인정보 기반 리타겟팅 시 GDPR/개인정보보호법 준수
- 10회 미만에서는 효과 미미, 20회 초과에서는 역효과 시작
